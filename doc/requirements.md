# 기능 요구사항

원본 요구사항: `[CRA_AI]반도체 시료 생산주문관리 프로젝트.pdf`

## 시스템 개요

가상 반도체 회사 S-Semi의 시료(Sample) 생산주문관리 시스템.
콘솔 기반 C++ 애플리케이션으로, 주문 접수부터 출고까지 전 과정을 관리한다.

## 역할 구조

| 역할 | 설명 |
|------|------|
| 고객 | 시료를 이메일로 요청 |
| 주문 담당자 | 고객 요청을 받아 주문서 작성·관리 |
| 생산 담당자 | 시료 등록, 주문 수신 후 승인 또는 거절 |

## 주문 상태 흐름

```
RESERVED
    │
    ├──[거절]──────────────→ REJECTED  (모니터링 제외)
    │
    └──[승인]──→ 재고 확인
                    │
                    ├──[재고 충분]──→ CONFIRMED ──→ RELEASE
                    │
                    └──[재고 부족]──→ PRODUCING ──→ CONFIRMED ──→ RELEASE
```

| 상태 | 의미 |
|------|------|
| RESERVED | 주문 접수 |
| REJECTED | 주문 거절 (모니터링 제외) |
| PRODUCING | 승인 완료, 재고 부족으로 생산 중 |
| CONFIRMED | 승인 완료, 출고 대기 중 |
| RELEASE | 출고 완료 |

## 메인 메뉴 구성

| 메뉴 | 기능 |
|------|------|
| 시료 관리 | 시료 등록 / 목록 조회 / 검색 |
| 시료 주문 | 고객 주문 접수 (RESERVED 생성) |
| 주문 승인/거절 | 승인·거절 처리, 재고 기반 상태 자동 전환 |
| 모니터링 | 상태별 주문 수, 시료별 재고 현황 |
| 생산라인 | 현재 생산 중 정보, FIFO 대기 큐 |
| 출고 처리 | CONFIRMED → RELEASE |

## 시료 속성

| 속성 | 설명 |
|------|------|
| ID | 고유 식별자 (예: S-001) |
| 이름 | 시료명 |
| 평균 생산시간 | min/ea |
| 수율 | 정상 시료 / 총 생산 시료 (0 < yield ≤ 1.0) |
| 재고 | 현재 보유 수량 |

## 생산량 계산

```
실생산량  = ceil(부족분 / (수율 × 0.9))
총생산시간 = 평균생산시간 × 실생산량
```
