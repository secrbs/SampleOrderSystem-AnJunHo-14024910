# Phase 1: 메인 UI

## 목표

MVC 뼈대를 구성하고, 전체 기능을 연결하는 메인 메뉴 루프와 시스템 현황 요약 화면을 구현한다.

## 구현 범위

### 시스템 현황 요약
메인 메뉴 진입 시 상단에 표시
- 등록 시료 수
- 총 재고 수량
- 전체 주문 수
- 생산라인 대기 수

### 메인 메뉴 루프
```
[1] 시료 관리
[2] 시료 주문
[3] 주문 승인/거절
[4] 모니터링
[5] 생산라인 조회
[6] 출고 처리
[0] 종료
```
- 잘못된 입력 시 오류 메시지 출력 후 재표시
- 각 메뉴 선택 시 해당 기능 진입 (Phase 2~7에서 구현)
- 이 Phase에서는 메뉴 선택 후 "준비 중" 메시지로 stub 처리

### MVC 뼈대
- `view/MainView` — 메인 메뉴 출력, 입력 처리
- `controller/MainController` — 메뉴 선택에 따른 분기
- `repository/SampleRepository` / `OrderRepository` — 파일 로드/저장 뼈대
- `main.cpp` — 앱 진입점, Repository 초기화 → MainController 실행

## 테스트 시나리오

| ID | 내용 | 예상 결과 |
|----|------|-----------|
| T1-01 | 메뉴 입력 0 | 프로그램 정상 종료 |
