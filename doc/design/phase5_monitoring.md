# Phase 5: 모니터링

## 목표

상태별 주문 현황과 시료별 재고 현황을 한눈에 확인하는 모니터링 기능을 구현한다.

## 구현 범위

### 주문량 확인
- REJECTED 제외, 상태별 주문 건수 표시
  - RESERVED / CONFIRMED / PRODUCING / RELEASE

### 재고량 확인
- 각 시료별 현재 재고 수량
- 재고 상태 판정

| 상태 | 조건 |
|------|------|
| 고갈 | 재고 = 0 |
| 부족 | 0 < 재고 < CONFIRMED 주문 합산 수량 |
| 여유 | 재고 ≥ CONFIRMED 주문 합산 수량 |

## 테스트 시나리오

| ID | 내용 | 예상 결과 |
|----|------|-----------|
| T5-01 | 상태별 주문 수 집계 (REJECTED 제외) | 올바른 건수 반환 |
| T5-02 | 주문 없는 상태는 0건 표시 | 0 반환 |
| T5-03 | 재고 0 → 고갈 | DEPLETED 반환 |
| T5-04 | 재고 < CONFIRMED 합계 → 부족 | SHORT 반환 |
| T5-05 | 재고 ≥ CONFIRMED 합계 → 여유 | SURPLUS 반환 |
